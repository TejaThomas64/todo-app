# Task Manager App (Full Stack)

A full-stack task management web application built with React, Node.js, Express, and Supabase authentication.

## Features
- User signup and login with Supabase Auth
- Create, update, delete tasks
- Subtasks support
- Due date tracking
- Task status filtering (All / Active / Completed)
- Email reminder notifications using scheduled cron jobs

## Tech Stack
- React (Vite)
- Node.js
- Express
- Supabase (Auth + Database)
- PostgreSQL

## Project Structure
frontend/ → React client  
backend/ → Express API server  

## Note
This is a learning project built to understand full-stack development, authentication flows, and backend scheduling concepts.
